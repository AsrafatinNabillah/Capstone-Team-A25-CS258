# Customer Segmentation For Personalized Retail Market

Perusahaan ritel online sering menghadapi tantangan dalam menentukan strategi promosi yang tepat. Banyak kampanye masih dilakukan secara massal tanpa mempertimbangkan perbedaan perilaku dan nilai pelanggan. Hal ini menyebabkan promosi tidak efektif, biaya pemasaran meningkat, dan loyalitas pelanggan menurun. Untuk mengatasi masalah tersebut, dibutuhkan pendekatan berbasis data yang mampu menggambarkan karakteristik setiap pelanggan. Customer segmentation menggunakan data transaksi menjadi solusi penting karena dapat mengelompokkan pelanggan berdasarkan perilaku pembelian sehingga strategi pemasaran lebih efisien, personal, dan relevan.
